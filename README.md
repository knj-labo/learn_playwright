# learn_playwright

## Usage

move to the root of the project
```bash
$ cd learn_playwright && pnpm install
```

develop the project
```bash
$ pnpm run dev
```

run the tests with the `playwright` package
```bash
$ pnpm run test
```

run the tests will pass and the snapshots will be updated
```bash
$ pnpx playwright test --update-snapshots
```
